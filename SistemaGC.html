¿Cómo usar el sistema?

✕
1
Sube el archivo GC base El archivo .xlsm con PARTICIPANTES y CRONOGRAMA ya completados. Sin hojas Carga.
2
Verifica los datos base El sistema lee automáticamente los participantes y las fechas del cronograma.
3
Carga los CSV de Blackboard Un CSV por módulo. Puede tener registros de múltiples sesiones juntos.
4
Descarga el Excel actualizado Los rangos se escriben directamente en las celdas. No hay fórmulas intermedias.
ℹ Regla: 0–60 min = rango 1 · 61–120 min = rango 2 · más de 120 min = rango 3
⚠ El CSV puede incluir varias sesiones juntas. El sistema detecta automáticamente el formato de fecha (DD-MM o MM-DD) comparando contra el cronograma.
