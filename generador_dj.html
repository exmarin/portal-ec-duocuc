¿Cómo funciona este sistema?
Generador de Declaraciones Juradas — Franquicia SENCE
1

Subir el PDF de inscripción OTIC

Carga la Carta de Inscripción de Participantes emitida por la OTIC (ej. OTIC SOFOFA). El sistema lee el PDF localmente y extrae automáticamente el nombre del curso, código SENCE, fechas y la nómina completa de participantes.

2

Subir la plantilla Word

Carga el archivo Modelo_DJ.docx. Este documento debe contener los siguientes marcadores de texto que serán reemplazados automáticamente:

Marcadores en la plantilla

Nombre completo→Nombre del participante
Rut,→RUT individual (ej. 21874113-4,)
Nombre del curso→Nombre completo del curso
CODIGO SENCE→Código SENCE extraído del PDF
fecha de inicio→Fecha de inicio (DD.MM.YYYY)
fecha de termino→Fecha de término (DD.MM.YYYY)
Nombre y rut→Firma: nombre + RUT
3

Revisar y corregir los datos

Antes de generar, puedes editar cualquier campo: nombre del curso, fechas, o los datos de cada participante. También puedes agregar participantes manualmente con el botón correspondiente.

4

Generar y descargar

El sistema genera un archivo .docx por cada participante. Puedes descargarlos uno a uno o todos juntos como un archivo ZIP. Todo el proceso ocurre en tu navegador, sin conexión a internet.

🔒 Todos los archivos se procesan 100% localmente. Ningún dato sale de tu equipo.

¿Reiniciar el proceso? Se perderán los datos actuales.
